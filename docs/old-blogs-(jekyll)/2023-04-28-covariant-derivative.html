    Koszul Connection and Covariant Derivative

    Jul 16, 202411 min read

    Good Resources

    • Wiki: associated bundle, Koszul connection
    • Frederic Schuller Lectures 19-25, specifically 23 for this blog
    • Nakahara Chapter 10.4

    We mostly follow the notation in Schuller Lecture 25.

    As part 2 of my series building towards “Twisting Supersymmetry” (see part 1 here), I will talk about connections on associated bundles induced by connections on principal bundle. We actually see this very often: U(1) gauge theory (electromagnetism) will often have a scalar field (scalar QED) or a Dirac spinor field (QED) charged under U(1). If one inspects the covariant derivative in scalar QED, it actually arises from the connection on the complex line bundle (C-bundle) induced by the connection on the U(1)-principal bundle. And for QED, it is the connection on the K⊕Kˉ vector bundle (Dirac spinor bundle) induced by the connection on the U(1)-principal bundle.

    Associated Bundle

    Loosely speaking, an associated bundle (associated to some G-principal bundle) is a way to attach to each point in x∈M a fiber π−1x≅F homeomorphic to a topological space F on which G acts on. This fiber could be a Lie group G itself (so a G-principal bundle is technically associated to itself), but this definition also allows for vector spaces, which means we can have representations (ρ,V) of G as fibers too.

    The formal definition (wiki) is considering the direct product modulo an equivalence relation

    ​PV​:=P×ρ​V:=(P×V)/∼(p,v)∼(p∙R​g,g−1∙L​v)​

    Intuition: it may initially seem weird that we are considering all pairs, but it turns out this equivalence relation basically collapses the entire G fiber of P to a single point in the base manifold M. So the only “degree of freedom” left is the V. So effectively we are attaching a copy of V to every point in the base manifold M.

    Covariant Derivative of Associated Bundle Section (Teaser)

    This is the end goal: given a vector X∈Tp​M and a section in a vector bundle σ:M→PV​ (we need vector bundle structure because otherwise subtraction won’t make sense in the fiber), we want to rigorously calculate the covariant (directional) derivative of σ wrt X direction

    ∇X​σ∣p​∈π−1(p)≅V

    We want this covariant derivative to satisfy various properties, which we can check later once we have defined it. For f∈C∞(M), T,S∈TM

    1. ∇fT+S​σ=f∇T​σ+∇S​σ
    2. ∇T​σ1​+σ2​=∇T​σ1​+∇T​σ2​
    3. ∇T​(fσ)=T(f)σ+f∇T​σ

    The definition that we work towards is the following: for a local section s:U⊂M→PV​, a local trivialisation φ:U⊂M→P, the exterior covariant derivative on P, and a V-valued function ϕ:P→V, a g-valued connection one-form ω:P→g

    ∇T​s:=(φ∗Dϕ)(T)=ds(T)+(φ∗ω)(T)∙dρ​s

    It is a definition dump for now but let’s explain the details.

    Exterior Covariant Derivative on P

    Definition: For an any-valued n-form f on Ωn(P),

    Df:=df∘hor

    Special case 1: if ω is g-valued connection one-form (n=1), for any X,Y∈TP,

    Dω(X,Y):=dω(hor(X),hor(Y))

    Special case 2: if ϕ is a F-valued function on P (n=0), for any X∈TP,

    Dϕ(X):=dϕ(hor(X))

    Correspondence between F-valued functions on P & global sections of associated bundle PF​

    One can define the covariant derivative using parallel transport on the associated vector bundle PV​ but that definition is difficult to work with. Instead we seek to define things by using the principal G-bundle P and defining objects on it, then pulling them over to the associated bundle PV​.

    We are going to work with a general fiber F (not necessarily a vector bundle).

    The first thing to establish is the one-to-one correspondence between (global) sections σ on PV​ and G-equivariance F-valued functions ϕ on P. G-equivariance means ϕ(p∙R​g)=g−1∙L​ϕ(p)

    The correspondence from σ to ϕσ​ is

    ϕσ​:Ppwhere ip​:Ff​→F↦ip−1​(σ(π(p)))→PF​↦[(p,f)]​

    One can check (watch Schuller Lec 25) that this ϕ is G-equivariant

    The other direction from ϕ to σϕ​ is

    σϕ​:Mx​→PF​↦[(p,ϕ(p))] for any p​

    One can check (watch Lec 25) that ϕσϕ​​=ϕ and σϕσ​​=σ

    Exterior Covariant Derivative of ϕ:P→V

    Only if we set the fiber F to be some finite dimensional vector space V (isomorphic to Rdim V), a (left) representation of G, then the G-equivariance condition

    ϕ(p∙R​exp(At))=exp(−At)∙ρ​ϕ(p)∀t∈R,A∈g

    can be differentiated (because the vector space has an addition). So fixing A∈g and differentiating wrt t,

    dtd​ϕ(p∙R​exp(At))​t=0​dϕ(XpA​)​=dtd​exp(−At)∙ρ​ϕ(p)​t=0​=−A∙dρ​ϕ(p)∈Tϕ(p)​V​

    where dϕ:Tp​P→Tϕ(p)​V is the pushforward of ϕ:P→V. The ∙dρ​:g×Tv​V→Tv​V action is the pushforward of the Lie group representation. Actually to be more precise it is the restriction of the pushforward to the tangent space at the identity of the group. To spell it out, we take the pushforward of ρ:G×V→V, namely dρ:Tg​G×Tv​V→Tρ(g,v)​V and restrict it to the subspace Tg​G⊃Te​G≅g. So the codomain becomes Tρ(e,v)​V=Tv​V.

    If we had a connection on P, then ω(XA)=A so we can rewrite the above as

    dϕ(XpA​)+ω(XpA​)∙dρ​ϕ(p)=0

    Now using the above, we can show that Dϕ:=dϕ∘hor is equals to

    Dϕ(Xp​)=dϕ(Xp​)+ω(Xp​)∙dρ​ϕ(p)

    by decomposing X=H+V and showing that the above holds for H,V separately.

    Pulling back Dϕ(X) to ∇π∗​X​σ

    We have a good expression for Dϕ:TP→Γ(PV​) (sections of the PV​ associated bundle), or pointwise a V-valued one-form Dϕp​:Tp​P→V. Suppose we have a local section φ:U→P, then we can pullback the Dϕ one-form to a one-form on the base manifold U⊂M. For a vector T∈Tx​U

    :=(φ∗Dϕ)(T)​​∇T​(ϕ∘φ)∇T​s​=(φ∗dϕ)T+(φ∗ω)(T)∙dρ​(φ∗ϕ)(x)=d(ϕ∘φ)T+Γ(T)∙dρ​ϕ(φ(x))=ds(T)+Γ(T)∙dρ​s(x)​

    where we ϕ∘φ≡s:U→V. The last line is just the usual definition we see for covariant derivative!

    One can check that all the properties below hold true given the above definition.

    For f∈C∞(M), T,S∈TM

    1. ∇fT+S​σ=f∇T​σ+∇S​σ
    2. ∇T​σ1​+σ2​=∇T​σ1​+∇T​σ2​
    3. ∇T​(fσ)=T(f)σ+f∇T​σ

    Examples

    General Relativity

    In General Relativity, the principal bundle is the frame bundle LM, with gauge group GL(d). The associated bundle is the tangent bundle TM. Since GL(d) is a matrix group, the action ∙dρ​ is just matrix multiplication. The section s:U→TM is a vector field. If there is a coordinate chart x:U→Rd, then the section can be decomposed as sj(x)∂j​ where sj:U→R are functions on U.

    ∇T​s=ds(T)+Γ(T)∙dρ​s(x)

    becomes

    (∇∂μ​​s)i​=[ds(∂μ​)]i+Γijμ​sj=∂μ​(si)+Γijμ​sj​

    To unpack what we just wrote above, note that Γμ​ is a matrix. If we represent s=sj∂j​ as a column vector with entries sj, then the action of Γμ​ on s is just matrix multiplication (Γμ​s)i=Γijμ​sj

    We have used property (1) to decompose ∇T​ into

    ∇Tμ∂μ​​=Tμ∇∂μ​​

    On the LHS, ∇∂μ​​s is a vector in Tx​M so we can extract components (∇∂μ​​s)i. Do note that this is NOT the same as ∇∂μ​​(si). The latter is interpreting si as functions, and the associated bundle would be the line bundle R. In our case we are dealing with an associated vector bundle, and elements living in the fiber are s(x)∈Tx​M.

    The ∂μ​(si) term is admittedly very ambiguous (my fault). We start from the definition of the pushforward

    ds(∂μ​):=∂μ​(_∘s)

    The function on U, _∘s:U→R, is eaten by ∂μ​∈Tx​U to yield a real number. But this ds(∂μ​) has a blank slot _, and is waiting for a function on TM.

    The ds(∂μ​) eats a function on TM and outputs a real number. In other words, it is a vector on TM (meaning it lives in TTM, yea!). ∇∂μ​​s on the LHS is a vector on TM too. All that remains to settle are the components: to show that

    [ds(∂μ​)]i=[∂μ​(_∘s)]i=!∂μ​(si)

    Actually this doesn’t make sense, because i goes from 1 to d on the RHS, whereas the dimension of the ds(∂μ​) vector is 2d. The resolution to this is to note that ds(∂μ​) is a horizontal vector since d(π∘s)=idTM​, so it has effective dimension d. Since d is a linear functional, (ds)i=d(si). Then d(si)∂μ​ is just ∂μ​(si) by definition. QED

    I’m sorry to leave you on the edge but I do not know how to make the above argument more rigorous. It matches with the GR definition of covariant derivative so I’m sure there is some valid explanation. I think it has to do with the isomorphism Tv​V≅V but I can’t make this precise.     Scalar Electrodynamics

    We switch to the convention that the exp map from Lie algebra to Lie group has a factor of i, so exp(ig)=G. This is so that the Lie algebra is real to match physics convention.

    The section s:M→C is the complex line bundle. The action ρ:G→GL(C) is given by complex rotation exp(iλ)∙ρ​ϕ=exp(iλ)ϕ. The pushforward of that action is ∙dρ​:u(1)→GL(C), given by complex multiplication λ∙dρ​ϕ=iλϕ. Γ is the electromagnetic 4-potential A=Aμ​dxμ, with Aμ​∈R≅g.

    ∇T​s=ds(T)+Γ(T)∙dρ​s(x)

    becomes

    D∂μ​​ϕ​=dϕ(∂μ​)+Aμ​∙dρ​ϕ=∂μ​ϕ+iAμ​ϕ​

    If we had chosen the action to be exp(iλ)∙ρ​ϕ=exp(−iλ)ϕ then we would have arrived at −iAμ​ϕ instead of + (a more common convention). It is just a matter of convention / choice of action.

    The Dirac Electrodynamics version is not too different since U(1) just acts on individual components (scalar multiplication as opposed to matrix multiplication) of the Dirac spinor.

    Gauge Transformations

    You hear this in GR: “the covariant derivative transforms like a tensor”. You also see things like Dμ′​U(x)=U(x)Dμ​ in electrodynamics. It turns out this is unified under the following

    ∇′Ω−1s​=d(Ω−1s)+(Ω−1ΓΩ+Ω−1dΩ)Ω−1s=dΩ−1s+Ω−1ds+Ω−1Γs+Ω−1dΩΩ−1s=dΩ−1s+Ω−1ds+Ω−1Γs−dΩ−1s=Ω−1(ds+Γs)=Ω−1∇s​

    where we used dΩ−1Ω+Ω−1dΩ=d(Ω−1Ω)=0.

    I find it slightly odd that s transforms under Ω−1 (from the left). I could have made a mistake, or maybe it’s just interpreted as a right action (left vs right differs under composition of actions).
